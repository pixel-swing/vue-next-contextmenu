# vue-next-contextmenu

基于 vue-next 3.0.0-beta.7 的 contextmenu 组件

## 概览

暴露如下组件：

- `ContextMenu`
- `ContextMenuItem`
- `ContextMenuSub`
- `ContextMenuGroup`

## 安装

### Prerequisites
- Node & NPM 

### Install
```sh
npm install 
```

## 使用

### Develop
```sh
# run dev server at localhost:8080
npm run dev
```
### Build
```sh
# transpile js for deployment
npm run build
```
